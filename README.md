# SGP Project

This is a full-stack college project.

## Project Structure

sgp/
├── frontend-react/   (React frontend)
├── backend/          (Node.js backend)
└── README.md

## Frontend
- React.js
- Folder: frontend-react
- Run using: npm start

## Backend
- Node.js + Express
- Folder: backend
- Run using: node app.js

## Database
- Not implemented yet (planned)

## How to Run Project

### Frontend
cd frontend-react
npm install
npm start

### Backend
cd backend
npm install
node app.js
